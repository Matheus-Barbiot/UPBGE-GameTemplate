Bem vindo ao UPBGE-GameTemplate. Criado por mim, Matheus Barbiot.

Este template foi criado para ser usado no programa UPBGE 0.2.5, um fork do antigo bge da versão 2.7 do blender

Aproveite este template e comece a criar seus jogos sem se preocupar com criação de Menus, pois o template vem como tudo pronto para você, com menu principal, configurações, creditos e pause. Todos estes podendo ser totalmente personalizados por você. Também há outras cenas como 'game' que apresenta um ambiente pronto para a criatividade e criação. Ou '_intro' que é por onde você deve extrair seu jogo do motor para evitar erros com as configurações.
